#  חלק ג – הסבר תמציתי לכל שאילתה

##  שאלה 1  
**מציאת שלשות מספרים שונים שסכום הערכים שלהם שווה ל־32.**  
בוצע JOIN על אותה טבלה שלוש פעמים תוך בדיקה שהערכים שונים, וסינון לפי סכום.

##  שאלה 2  
**מציאת קומבינציות של שלוש רשומות (מאותה טבלה) שסכומן שווה לערך קלט.**  
בוצע חיבור של כל קומבינציה אפשרית תוך שימוש ב־`CROSS JOIN` עם סינון של סכום.

##  שאלה 3  
**מציאת שלשות עם אותו סכום – והצגה של הזמן הממוצע לשליחה.**  
בוצע חיבור של שלוש בקשות עם אותו סכום עדיפויות, וחישוב ממוצע זמן שליחתן.
