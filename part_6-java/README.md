# Part 6 – Invoice Processing (Java)

בחלק זה ממומשת לוגיקת ניהול חשבוניות בשפת Java.

## קבצים עיקריים:

- **Invoice.java**  
  מגדיר מחלקה המייצגת חשבונית, כולל שדות כמו תאריך הוצאה, תאריך יעד, סכום לתשלום, סכום ששולם, ודגל האם שולם. המחלקה כוללת גם תמיכה בשדה `daysLate` עבור איחור בתשלום.

- **Main.java**  
  מדגים שימוש במחלקת `Invoice`: יצירת חשבוניות, חישוב איחור בתשלום וביצוע עיבוד בסיסי. הקוד מיועד להרצה קונסולית פשוטה ולמדגים את זרימת העבודה.

## הערות:
- הקוד נכתב ללא שימוש בתלויות חיצוניות.
- ניתן להריץ את הקוד בכל סביבת Java (גרסה 8 ומעלה).
