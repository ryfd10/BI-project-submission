# הגשת פרויקט BI – מטלת בית

## סקירה כללית

ריפוזיטורי זה כולל את הפתרונות המלאים שלי למטלת הבית בתחום BI (בינה עסקית).  
המטלה כללה מספר שאלות המבוססות על SQL, וכן חלק קצר בשפת Java.

כל חלק בפרויקט מסודר בתיקייה נפרדת, כאשר בכל תיקייה ניתן למצוא את קובצי הקוד המתאימים, יחד עם קובץ README שמסביר בקצרה את הלוגיקה מאחורי כל פתרון.

---

## מבנה הפרויקט

- `part_1` עד `part_5`: פתרונות לשאלות SQL.
  - כל תיקייה כוללת קובצי `.sql` מסודרים לפי השאלות.
  - לכל חלק קובץ `README.md` שמכיל הסבר קצר על כל שאילתה.

- `part_6-java`: מימוש בשפת Java של תרגיל בנושא ניהול חשבוניות.
  - כולל את הקבצים `Main.java`, `Invoice.java` ו־`README.md`.

- `documentation`: תיקייה אופציונלית שמכילה את קובץ ה־PDF של הדו"ח הכולל (למשל: צילומי מסך, תוצרי שאילתות, ועוד).

---

## הערות כלליות

הקפדתי לנסח פתרונות ברורים, יעילים ומבוססי לוגיקה, תוך שמירה על תיעוד קצר וממוקד.  
כל השאילתות נבדקו ונוסו על בסיס הנתונים שסופק במטלה.
