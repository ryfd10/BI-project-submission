# היפוך מחרוזת באמצעות CTE רקורסיבי

קוד זה מבצע היפוך של מחרוזת נתונה בלי להשתמש בפונקציה המובנית `REVERSE`, אלא באמצעות לוגיקת רקורסיה בתוך CTE.

## לוגיקת הפתרון:

- נגדיר משתנה מחרוזת (`@txt`) ואינדקס שמתחיל באורך המחרוזת.
- נבנה CTE רקורסיבי שמתחיל מהסוף של המחרוזת ומצרף תו־אחר־תו לתוצאה.
- בכל איטרציה, האינדקס קטן ב־1 עד שמגיע ל־0.
- בסיום, נשלפת השורה שבה ה־index הגיע ל־0 – והיא מכילה את המחרוזת ההפוכה.
